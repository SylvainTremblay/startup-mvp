MVPBUILDFAST.COM (STARTUP MVP)
or BuildMVPFast.com 


Allow Startup to quickly and easily generate a functional MVP application.
Users can select from sets of screen type, templates and entity models using a
WISIWIG Web editor and create the screen prototypes using Fabric.js 
http://fabricjs.com/fabric-intro-part-1
https://github.com/adrianhajdin/figma_clone?tab=readme-ov-file#tech-stack

Editor could be Sketch like instead of too formal.
- have tutorials

- Subscription based and have tracking data for their MVP so they are locked in

Once their screen prototypes meat their MVP requirements, they can
generate and see the mobile app executed in their web browser.
They can then decide to download and publish the applicatoin.

AI: free text where they describe their ideas and we generate wireframe from it

- support for advanced customization
- subscription based model or one time
- service to store the prototype so they can come back and modify it
- integration with external APIs 
- integration with pre-defined APIs
- APIs generation
- integration with MVP database.
- analytics integration 
- payment integration
- login integration with google, facebook, etc.
- MFA login
- Capchat
- Automatic app submission to app store
